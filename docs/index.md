# Sonolus.py
Sonolus.py is a Python library for creating Sonolus engines.

## Installation
Sonolus.py is available on PyPI and can be installed using a package manager like pip.

=== "pip"
    ```bash
    pip install sonolus.py
    ```
=== "uv"
    ```bash
    uv add sonolus.py
    ```
